Essential reading for Arch beginners

Arch is a do-it-yourself distribution — you assemble the system, and the ArchWiki is your manual. These are the pages worth reading first. ArcoLinux Nemesis runs on top of an Arch install you set up with these.

The golden rule

Before every big system update, skim the Arch news. Breaking changes that need manual intervention are announced there first — reading it saves you from a broken boot.
